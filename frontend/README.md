# Uteach Audio Summary Frontend

Modern React + TypeScript frontend for the Uteach Audio Summary application.

## Features
- **Audio Capture**: Record via browser or upload files.
- **IndexedDB Recovery**: Sessions persist through reloads.
- **AI Configuration**: Multiple summary styles (Paragraph, Table).
- **PDF Export**: Modern styled PDF generation.

## Setup
1. `cd frontend`
2. `npm install`

## Running
`npm run dev`
